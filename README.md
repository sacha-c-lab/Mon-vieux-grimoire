# Mon-vieux-grimoire

## Installation

**Backend**
```
cd backend
npm install
node server.js
```

Créer un fichier `.env` dans le dossier `backend/` avec :
```
TOKEN_SECRET=RANDOM_TOKEN_SECRET
MONGODB_URI=mongodb+srv://sacha:204910@cluster0.stndrnd.mongodb.net/
```

**Frontend**
```
cd P7-Dev-Web-livres
npm install
npm start
```
